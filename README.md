# Implementación Kafka

## Uso 
```sh
git clone https://github.com/pixal-lab/SD_T2
cd SD-T2
docker compose up -d
```

### ejecutar consumidor

```sh
docker exec -it consumer_kafka bash
python3 gremio.py
```

### ejecutar productor

```sh
docker exec -it producer_kafka bash
python3 Escenario.py
```
